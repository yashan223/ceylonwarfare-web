HOW TO INSTALL COD4 MP

Follow this comprehensive step by step multiplayer setup guide to get Call of Duty 4 Multiplayer running, patched with CoD4x, and optimized.

Step 01

Download Call of Duty 4 MW (Clean Game Files)

Before installing patches, you will need a clean base game folder containing the core Call of Duty 4: Modern Warfare assets. This version is lightweight, clean, and pre-extracted for multiplayer purposes.

  1. Download the clean base multiplayer game files archive using the button below.
  2. Extract the contents of the zip file to your preferred games folder (for example, C:\Games\Call of Duty 4).
Important: This clean game version does not require you to run any installers. Just extract the archive and proceed to the next step.
download Download COD4 Base (Clean)
Step 02

Get CoD4X Multiplayer Patch

CoD4X is a community-driven multiplayer patch that updates your game client to version 1.8. It patches crucial security vulnerabilities, fixes code bugs, and enables automatic background downloading of custom maps, mods, weapon skins, and killcam audio when connecting to server networks like Ceylon Warfare.

  1. Download the CoD4X installer executable directly using the button below (hosted by cod4x.ovh).
  2. Move the downloaded installer executable file into your main Call of Duty 4 game directory (where iw3mp.exe is located).
  3. Run the installer executable to apply the patch files and update the client.
download Download CoD4X Installer open_in_new Official cod4x.ovh website
Step 03

Provide Multiplayer Keycode

To access multiplayer, Call of Duty 4: Modern Warfare requires a CD key. While CoD4X bypasses global online check validation, the local game engine still demands a validly formatted keycode to allow server connections.

Use one of the pre-verified keycodes below. Click the copy icon next to a keycode to copy it to your clipboard:

DL2J-8PYW-4Q2M-Y47Q-885A
JW8G-4WPD-42J4-Y2QP-5DE7
4PME-DYQW-G8Q2-UWY4-90AE
2EPM-QWG8-Q2UW-Y490-6712
M8W4-2D4P-82GY-W4Q2-85D3
  1. Open the game using cod4x_mp.exe.
  2. Navigate to Options → Multiplayer Options → Enter CD Key.
  3. Paste your copied keycode into the field, then click Verify.
Step 04

Apply Machannoob Custom Config (CFG)

A professional player configuration (CFG) improves in-game visibility, optimizes multiplayer networking settings (rates/cl_maxpackets), sets custom client binds, enables fullbright maps, and increases FPS drastically.

We recommend downloading and installing the custom Machannoob CFG below:

  1. Download machannoob.zip.
  2. Extract the zip to retrieve the config_mp.cfg file.
  3. Copy the config_mp.cfg file and paste it into your profile folder. The exact path depends on your installation type:
CoD4x Path (Recommended):
Press Win + R, paste the following directory string, and hit enter:
%localappdata%\CallofDuty4MW\players\profiles\
Open your active profile's folder and paste the config_mp.cfg file inside (overwriting the existing one).
Vanilla Game Directory (Fallback):
If you are not running Local AppData profiles, place the configuration in:
[Your COD4 Main Folder]\players\profiles\[Your Profile Name]\config_mp.cfg
